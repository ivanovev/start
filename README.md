start
=====

Monitoring and control program for various devices.

How-to start:  
    git clone .../start.git  
    ./make.sh submodule add [hm|misc|...]  
    python startall.py  

Note: make.sh simply adds submodule to the source tree, it's also possible to add modules manually

Requirements:  
    python3  
    matplotlib  
    numpy  
    pyserial
